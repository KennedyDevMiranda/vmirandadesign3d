# Vmiranda3D — Landing Premium (Next.js + Tailwind)

## Rodar local
```bash
npm install
npm run dev
```

## Deploy na Vercel
- Conecte este repositório.
- Framework Preset: **Next.js**
- Root Directory: **/** (onde está o package.json)
- Build Command: `next build` (padrão)
- Output Directory: (deixe vazio)
